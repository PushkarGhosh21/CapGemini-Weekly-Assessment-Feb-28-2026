Project Overview
This repository contains a comprehensive collection of assessments and practice modules covering Core Java, SQL, and Hibernate ORM. It serves as a structured record of technical proficiency in backend development.

Repository Contents
1. Core Java Foundations (java/workspace/src)
A comprehensive repository of Java programming fundamentals, including:

Basic Syntax & Logic: Data types, operators, and control flow.

Language Features: String handling, Arrays, and Exception handling.

Advanced Concepts: Collections Framework, Multithreading, and I/O Streams.

Practice Labs: Hands-on solutions to common algorithmic problems and logic puzzles.

2. SQL & Database Design (sql)
Day-wise progression of database management exercises focused on MySQL:

Day 1-2: DDL (Data Definition Language) and DML (Data Manipulation Language) basics.

Day 3: Complex Queries including Joins, Subqueries, and Set operators.

Day 4: Database constraints, Aggregations, and Data Integrity practice.

3. Core Java OOP Assessment (oopsAssessment)
A console-based food ordering system designed to demonstrate mastery of Object-Oriented principles.

FoodItem (Abstract): Implementation of Abstraction and Encapsulation.

VegItem & NonVegItem: Demonstration of Inheritance and Method Overriding.

Order Logic: Polymorphic behavior for category-specific discounts and bill generation.

4. Hibernate ORM Assessment (com.hibernateassessment)
A Java application demonstrating persistent data management using the Hibernate framework.

Entity Mapping: Employee.java utilizing JPA annotations for ORM.

Session Management: Launch.java showcasing standard CRUD operations (Create, Read, Update, Delete).

Configuration: hibernate.cfg.xml for MySQL dialect settings and connection pooling.

Dependency Management: pom.xml configured for hibernate-core (8.0.0.Alpha1) and mysql-connector-j.

Tech Stack
Languages: Java, SQL

Frameworks: Hibernate ORM

Database: MySQL

Build Tool: Maven
